開發記錄

Web Reporter & Web App

  • 時間：2019

  • 相關工具：C#, ASP.NET Core 2.2, Bootstrap V4, Nginx, Ubuntu Server 18.04, ClosedXML

  • 描述：

    進到公司不到2個月，被分配到同事已經hold兩個月的需求，要系統以TipTop的資料客製化34個客戶格式的報表，涉及誇庫多角貿易，仔細了解需求後，公司現有的系統無法很有效率的滿足，因此提出引入Web系統的提案，並詳列幾個計劃及我為什麼需要使用這些技術的原因。先取得同事的共識後，再取得部門主管中立意見，隨即開始架構並完成功能。

    計劃開始2天內完成基本架構，第3天完成第一個報表程式，第二週時正好製造單位有個線上SOP需求交由我評估，評估後決定以Web的方式最能滿足該需求，隨即提供詳細的系統設計文件取得user同意後於1週內完成上線。

    目前系統在Linux(Ubuntu Server 18.04 LTS)上順利運行，涉及業管、製造、技術等單位。
